1) 다운로드 : 데이터나 파일을 인터넷 또는 원격 서버에서 컴퓨터나 장치로 정송

2) LINK : 웹 페이지나 리소스를 연결하는 하이퍼링크를 통해 접근할 수 있는 서비스

3) REST : Representational State Transfer의 약자로, 웹 서비스 설계 스타일 중 하나,
          HTTP 프로토콜을 사용해서 데이터 전송

4) SOAP : Simple Object Access Protocal, xml 기반 메세지 프로토콜

5) Rss/Atom : 웹사이트의 데이터를 업데이터를 신속하게 한 후 사용자에게 피드백

6) LOD : Linked Open Data / 웹에서 구조화된 데이터를 링크드 데이터의 형식으로 공개하는 것

[구조화된 데이터 = 정형]
 - 형식이 rwo, column으로 된 데이터, 스키마고정, 검색용이 - sql, csv 등 / 고객정보, 금융거래기록, 인벤토리 시스템

[반구조화된 데이터 - 반정형]
 - 일정한 구조로 되어 있지만 스키마는 고정이 아님 - 필드, 칼럼이 없음, 형태가 다양 - 태그나 마커를 사용함 - xml, json, yaml / 웹서비스 api응답, 구성파일, 로그파일

[비구조화된 데이터 - 비정형] 검색 x
 - 구조, 스키마, 형식 등이 없다, 다양한 형식 - 텍스트(워드, pdf), 이미지, 영상, 오디오 - 오픈 csv에서 관리 / 이메일, 고객 리뷰 피드백, 비디오 스트리밍 데이터
