HTML5 = HTML[문서] + CSS[스타일] + JS[액션=이벤트]
 - HTML[문서]의 기본태그 = 웹 페이지 구조 + 컨텐츠 정의(문서의 제목, 단락, 링크, 이밎, 목록 등)

1) 문서 기본 태그
 <!DOCTYPE html> : HTML5로 선언
<html lang="en"> : HTML root 태그
<head> : 문서의 메타데이터(제목, 스타일, 정보 등)
<meta charset="UTF-8">
<title> : 문서 제목
<body> : 문서 본문

2) 메타태그
<meta charset="UTF-8"> : 문서 메타 정의(문자 인코딩, 저자, 설명 등)
<link> : 외부 문서 링크(.css)
<style> : 문서의 스타일 정의
<script> : 자바스크립트 코드를 포함하거나 파일을 참조

3) 텍스트 콘텐츠 태그
<h1> ~ <h6> : 제목
<p> : 단락(문단태그) 정의
<br> : 줄바꿈
<hr> : 수평선 _주제 분리
<strong> : 중요한 문장
<em> : 강조
<samll> 중요하지 않은 텍스트
<mark> : 하이라이트 된 텍스트
<div> : 영역 태그

4) 목록 태크
<ul> : 순서 없는 목록
<ol> : 순서 있는 목록
<li> : 목록 항목
<dl> : 정의목록
<dt> : 정의 목록의 용어를 정의
<dd> : 정의 목록의 정의를 정의

5) 링크 및 미디어
<a> : 하이퍼링크
<img> : 이미지
<audio> 오디오
<video> : 비디오

6) 테이블 태크
<table> <tr> <td> <th> <tbody> <tfoot> <caption>

7) 폼태크

8) 임베딩 콘텐츠
<ifram> : 인라인 프레임
<embed> : 외부 대화형 컨텐츠
<object> : 외부 자원 프레임
<param> : 오브젝트 태그안에 매개변수

9) 시맨틱 태그
<header> : 문서의 섹션의 머리말
<nav> : 네비게이션 링크
<section> : 문서
<article> : 독립적인 컨텐츠
<aside> : 연간 컨텐츠
<footer> : 바닥글
<main> : 주요 컨텐츠
<figure> : 자립형 컨텐츠
<figcation> : 제목 정의
